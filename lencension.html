Launc Develop Adding a new product
To add a game to the Launc website, you need to click on the Create button in the menu on the left. Next you need to fill in the fields and pay attention to Requirements where you need to put the system requirements, if you do not know what to put, then leave the default values.
And for icons and screenshots it is recommended to use: google drive, yandex disk, ltdfoto.ru. To download zip file of the product, please use google drive with direct link (via service https://www.votix.ru/p/sozdat-pryamuyu-ssylku-fajla-google-disk.html), if you use oxy cloud and similar services, the product will be deleted.
To get a youtube id for the game there are a few steps to take: 1. Create and publish a video on the Youtube platform, 2. In the URL string you need to find the value after watch?v=, but this value cannot include &ab_channel= or ?t=1
Game description should be co-operative. It should say what the product is in general. Also do not need to put the actual version of the game, as it is written in a special field Version. But in the description you can write a list of version changes.
At the moment Launc Protection Dll for products is in development and you can help if you understand it, in report>idea
